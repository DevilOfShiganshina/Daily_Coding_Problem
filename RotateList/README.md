# Problem

Given a list of integers, rotate the list at a pivot k.

For example, given the list `[1, 2, 3, 4, 5]` and k = 3,\
return `[4, 5, 1, 2, 3]`

## Program Output

```
RotateList$ ./exe 
Original array: 1 2 3 4 5 
Rotated array: 4 5 1 2 3 
```

## Time Complexity of program
O(n) time\
O(1) space
